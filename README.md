# Tujar

## Project Description
Tujar is a platform that connects buyers and sellers, providing an easy-to-use interface for trading products and services.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/eldabyco-dotcom/tujar.git
   ```
2. Install dependencies:
   ```bash
   cd tujar
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Development Guide
- Ensure you are using Node.js version 14 or higher.
- Follow our coding standards and commit messages guidelines.
